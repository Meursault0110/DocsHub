## 1. Transiciones

Las transiciones en CSS permiten cambiar suavemente las propiedades de un elemento de un estado a otro.

<Tabs>
  <TabItem label="Sintaxis Básica">
    ```css
    .elemento {
      transition: propiedad duración función-de-tiempo retraso;
    }
    ```
  </TabItem>
  <TabItem label="Ejemplo">
    ```css
    .boton {
      background-color: blue;
      transition: background-color 0.3s ease-in-out;
    }
    .boton:hover {
      background-color: red;
    }
    ```
  </TabItem>
</Tabs>

<Aside type="note">
  Las transiciones son ideales para efectos simples activados por eventos como hover o focus.
</Aside>

## 2. Animaciones

Las animaciones en CSS ofrecen un control más detallado sobre las secuencias de movimiento y cambios de estilo.

<Tabs>
  <TabItem label="Definición de Keyframes">
    ```css
    @keyframes nombre-animacion {
      0% { /* estilos iniciales */ }
      50% { /* estilos intermedios */ }
      100% { /* estilos finales */ }
    }
    ```
  </TabItem>
  <TabItem label="Aplicación de Animación">
    ```css
    .elemento-animado {
      animation: nombre-animacion 5s ease-in-out infinite;
    }
    ```
  </TabItem>

## 3. Animaciones basadas en scroll

Las animaciones basadas en scroll crean efectos interactivos que responden al desplazamiento de la página.

<Steps>
  1. Definir los keyframes de la animación.
  2. Aplicar la animación al elemento deseado.
  3. Utilizar `animation-play-state` para controlar la animación basada en el scroll.
</Steps>

```css
.elemento-scroll {
  animation: aparecer 1s ease-out forwards;
  animation-play-state: paused;
}

@keyframes aparecer {
  from { opacity: 0; transform: translateY(50px); }
  to { opacity: 1; transform: translateY(0); }
}
```

```javascript
window.addEventListener('scroll', () => {
  const elementos = document.querySelectorAll('.elemento-scroll');
  elementos.forEach(elem => {
    const rect = elem.getBoundingClientRect();
    if (rect.top < window.innerHeight) {
      elem.style.animationPlayState = 'running';
    }
  });
});
```

<Aside type="caution">
  Las animaciones basadas en scroll pueden afectar el rendimiento si no se optimizan adecuadamente.
</Aside>

## 4. Rango de Animaciones

El rango de animaciones permite un control preciso sobre diferentes etapas de una animación mediante porcentajes.

```css
@keyframes cambio-color {
  0% { background-color: red; }
  25% { background-color: yellow; }
  50% { background-color: green; }
  75% { background-color: blue; }
  100% { background-color: purple; }
}

.elemento-color-cambiante {
  width: 100px;
  height: 100px;
  animation: cambio-color 5s infinite;
}
```

<Badge variant="success">Pro Tip</Badge> Utiliza `animation-timing-function: steps(5)` para crear una animación que cambie de color en intervalos discretos.
