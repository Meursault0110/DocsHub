## Introducción a CSS

CSS (Cascading Style Sheets) es un lenguaje de estilo utilizado para describir la presentación de un documento escrito en HTML o XML. CSS describe cómo los elementos deben ser renderizados en pantalla, en papel, o en otros medios.

<Card title="¿Por qué es importante CSS?" icon="question">
  CSS permite separar el contenido de la presentación, lo que resulta en:
  - Mayor flexibilidad en el diseño
  - Mejor mantenimiento del código
  - Capacidad para adaptar el diseño a diferentes dispositivos
</Card>

## Selectores Básicos

Los selectores son patrones utilizados para seleccionar los elementos que quieres estilizar.

<CardGrid>
  <Card title="Selector de Elemento" icon="document">
    Selecciona todos los elementos de un tipo específico.
    ```css
    p {
      color: blue;
    }
    ```
  </Card>
  <Card title="Selector de Clase" icon="puzzle">
    Selecciona elementos con una clase específica.
    ```css
    .mi-clase {
      font-size: 16px;
    }
    ```
  </Card>
  <Card title="Selector de ID" icon="target">
    Selecciona un elemento con un ID específico.
    ```css
    #mi-id {
      background-color: yellow;
    }
    ```
  </Card>
</CardGrid>

## Relleno y Margen (Padding y Margin)

<Steps>
  1. **Padding**: Espacio entre el contenido y el borde.
  2. **Margin**: Espacio fuera del borde.
</Steps>

```css
.elemento {
  padding: 10px;
  margin: 20px;
}
```

## Bordes (Border)

Los bordes pueden personalizarse en estilo, ancho y color.

```css
.elemento {
  border: 1px solid black;
  border-radius: 5px;
}
```

## Tamaño de la Caja (Box sizing)

<Aside type="tip">
  Usa `box-sizing: border-box;` para incluir padding y border en el ancho total del elemento.
</Aside>

```css
* {
  box-sizing: border-box;
}
```

## Unidades (CSS Units)

CSS proporciona varias unidades para especificar tamaños.

<Tabs>
  <TabItem label="Absolutas">
    - `px`: Píxeles
    - `cm`: Centímetros
    - `mm`: Milímetros
    - `in`: Pulgadas
    - `pt`: Puntos
    - `pc`: Picas
  </TabItem>
  <TabItem label="Relativas">
    - `em`: Relativo al tamaño de fuente del elemento
    - `rem`: Relativo al tamaño de fuente del elemento raíz
    - `%`: Porcentaje relativo al elemento padre
    - `vw`: 1% del ancho de la ventana gráfica
    - `vh`: 1% del alto de la ventana gráfica
  </TabItem>
</Tabs>

## Fondos (Background)

CSS ofrece varias propiedades para controlar los fondos de los elementos.

```css
.elemento {
  background-color: #f0f0f0;
  background-image: url('fondo.jpg');
  background-repeat: no-repeat;
  background-position: center;
  background-size: cover;
}
```

## Sombras (Shadows)

Las sombras pueden aplicarse a texto y elementos.

```css
.texto-con-sombra {
  text-shadow: 2px 2px 4px rgba(0,0,0,0.5);
}

.caja-con-sombra {
  box-shadow: 0 4px 8px 0 rgba(0,0,0,0.2);
}
```

<Aside type="caution">
  El uso excesivo de sombras puede afectar el rendimiento. Úsalas con moderación.
</Aside>
