## Selectores Avanzados

Los selectores son la base de CSS. Permiten apuntar a elementos HTML específicos para aplicar estilos.

<CardGrid>
  <Card title="Selectores de Atributo" icon="puzzle">
    `[attr]`: Selecciona elementos con el atributo attr.
    `[attr="value"]`: Selecciona elementos donde attr es exactamente "value".
    `[attr~="value"]`: Selecciona elementos donde attr contiene "value" como palabra completa.
  </Card>
  <Card title="Selectores de Relación" icon="random">
    `A > B`: Selecciona B que son hijos directos de A.
    `A + B`: Selecciona B que es el hermano adyacente de A.
    `A ~ B`: Selecciona B que son hermanos de A.
  </Card>
</CardGrid>

## Herencia, Cascada y Especificidad

Estos conceptos son cruciales para entender cómo se aplican los estilos en CSS.

<Steps>
  1. **Herencia**: Algunas propiedades CSS se heredan de los elementos padres a los hijos.
  2. **Cascada**: Determina qué reglas se aplican cuando hay múltiples reglas conflictivas.
  3. **Especificidad**: Determina qué regla tiene prioridad basándose en la precisión del selector.
</Steps>

<Aside type="note">
  La especificidad se calcula basándose en la combinación de selectores utilizados. Los selectores de ID tienen la mayor especificidad, seguidos por las clases y los atributos, y finalmente los elementos.
</Aside>

## Pseudo-clases y Pseudo-elementos

Las pseudo-clases y pseudo-elementos permiten aplicar estilos basados en estados específicos o partes de un elemento.

<Tabs>
  <TabItem label="Pseudo-clases">
    - `:hover` - Cuando el mouse está sobre el elemento
    - `:active` - Cuando el elemento está siendo activado
    - `:focus` - Cuando el elemento tiene el foco
    - `:first-child` - El primer hijo de su padre
    - `:nth-child(n)` - El enésimo hijo de su padre
  </TabItem>
  <TabItem label="Pseudo-elementos">
    - `::before` - Inserta contenido antes del contenido del elemento
    - `::after` - Inserta contenido después del contenido del elemento
    - `::first-line` - Selecciona la primera línea de texto
    - `::first-letter` - Selecciona la primera letra del texto
  </TabItem>
</Tabs>

## Metodología BEM

BEM (Block, Element, Modifier) es una metodología de nomenclatura para clases CSS que ayuda a crear componentes reutilizables y código mantenible.

<Card title="Estructura BEM" icon="puzzle">
  - **Bloque**: Componente independiente (ej. `card`)
  - **Elemento**: Parte de un bloque (ej. `card__title`)
  - **Modificador**: Variación de un bloque o elemento (ej. `card--featured`)
</Card>

Ejemplo de uso de BEM:

```css
.card { /* Estilos para el bloque card */ }
.card__title { /* Estilos para el elemento título dentro de card */ }
.card--featured { /* Estilos para la variante destacada de card */ }
```

<Aside type="tip">
  BEM ayuda a evitar conflictos de especificidad y hace que el código sea más predecible y fácil de mantener.
</Aside>

## Display

La propiedad `display` controla cómo se muestra un elemento en el flujo del documento.

    - Ocupa todo el ancho disponible
    - Comienza en una nueva línea
  </Card>
  <Card title="inline">
    - Ocupa solo el espacio necesario
    - No comienza en una nueva línea
  </Card>
  <Card title="inline-block">
    - Combinación de inline y block
    - Permite establecer ancho y alto
  </Card>
  <Card title="flex">
    - Crea un contenedor flexible
    - Permite alinear y distribuir elementos hijos
  </Card>
  <Card title="grid">
    - Crea un sistema de cuadrícula
    - Permite diseños complejos en dos dimensiones
  </Card>
</CardGrid>

## Posicionamiento

CSS ofrece varias opciones para controlar el posicionamiento de los elementos.

<Tabs>
  <TabItem label="Relativo">
    El elemento se posiciona en relación a su posición normal.
    ```css
    .elemento { position: relative; top: 10px; left: 20px; }
    ```
  </TabItem>
  <TabItem label="Absoluto">
    El elemento se posiciona en relación al ancestro posicionado más cercano.
    ```css
    .contenedor { position: relative; }
    .elemento { position: absolute; top: 0; right: 0; }
    ```
  </TabItem>
  <TabItem label="Fijo">
    El elemento se posiciona en relación a la ventana del navegador.
    ```css
    .elemento { position: fixed; bottom: 20px; right: 20px; }
    ```
  </TabItem>
  <TabItem label="Sticky">
    El elemento alterna entre relativo y fijo según la posición de scroll.
    ```css
    .elemento { position: sticky; top: 0; }
    ```
  </TabItem>
</Tabs>

## Transiciones

Las transiciones permiten cambiar suavemente los valores de las propiedades CSS a lo largo del tiempo.

```css
.elemento {
  transition: property duration timing-function delay;
}
```

<CardGrid>
  <Card title="Property">
    La propiedad CSS a animar (e.g., `color`, `opacity`, `transform`)
  </Card>
  <Card title="Duration">
    La duración de la transición en segundos o milisegundos
  </Card>
  <Card title="Timing Function">
    La curva de aceleración de la transición (e.g., `ease`, `linear`, `cubic-bezier`)
  </Card>
  <Card title="Delay">
    El tiempo de espera antes de que comience la transición
  </Card>
</CardGrid>

Ejemplo:

```css
.button {
  background-color: blue;
  transition: background-color 0.3s ease;
}

.button:hover {
  background-color: red;
}
```

<Aside type="note">
  Las transiciones son una forma excelente de añadir interactividad sutil a tu sitio web sin necesidad de JavaScript.
</Aside>

## Overflow

La propiedad `overflow` controla cómo se maneja el contenido que excede las dimensiones de su contenedor.

<CardGrid>
  <Card title="visible">
    El contenido desborda el contenedor (comportamiento predeterminado)
  </Card>
  <Card title="hidden">
    El contenido que desborda se recorta y no es visible
  </Card>
  <Card title="scroll">
    Agrega barras de desplazamiento, incluso si no son necesarias
  </Card>
  <Card title="auto">
    Agrega barras de desplazamiento solo cuando son necesarias
  </Card>
</CardGrid>

```css
.contenedor {
  width: 300px;
  height: 200px;
  overflow: auto;
}
```

## Control de flujo del Texto

CSS proporciona varias propiedades para controlar cómo fluye el texto dentro de los elementos.

<CardGrid>
  <Card title="text-align">
    Alinea el texto horizontalmente (left, right, center, justify)
  </Card>
  <Card title="line-height">
    Establece la altura de las líneas de texto
  </Card>
  <Card title="text-indent">
    Establece la sangría de la primera línea de un bloque de texto
  </Card>
  <Card title="word-spacing">
    Ajusta el espacio entre palabras
  </Card>
  <Card title="letter-spacing">
    Ajusta el espacio entre caracteres
  </Card>
</CardGrid>

Ejemplo:

```css
.parrafo {
  text-align: justify;
  line-height: 1.5;
  text-indent: 20px;
  word-spacing: 2px;
  letter-spacing: 0.5px;
}
```

<Aside type="tip">
  Un buen control del flujo del texto puede mejorar significativamente la legibilidad de tu contenido.
</Aside>

## Object-fit y Object-position

Estas propiedades son útiles para controlar cómo se ajustan las imágenes y videos dentro de sus contenedores.

<Tabs>
  <TabItem label="object-fit">
    - `fill`: Estira la imagen para llenar el contenedor
    - `contain`: Ajusta la imagen manteniendo su relación de aspecto
    - `cover`: Cubre todo el contenedor, recortando si es necesario
    - `none`: No ajusta la imagen
    - `scale-down`: Usa el menor de `none` o `contain`
  </TabItem>
  <TabItem label="object-position">
    Controla la posición de la imagen dentro del contenedor.
    Ejemplo: `object-position: center top;`
  </TabItem>
</Tabs>

```css
.imagen {
  width: 300px;
  height: 200px;
  object-fit: cover;
  object-position: center;
}
```

## Outline (Contorno)

La propiedad `outline` crea un contorno alrededor de un elemento, fuera del borde.

```css
.elemento {
  outline: width style color;
}
```

<CardGrid>
  <Card title="width">
    Grosor del contorno (e.g., `2px`)
  </Card>
  <Card title="style">
    Estilo del contorno (e.g., `solid`, `dotted`, `dashed`)
  </Card>
  <Card title="color">
    Color del contorno
  </Card>
</CardGrid>

Ejemplo:

```css
.input:focus {
  outline: 2px solid blue;
}
```

<Aside type="note">
  A diferencia del `border`, el `outline` no afecta el layout de la página, lo que lo hace útil para elementos de enfoque.
</Aside>
