## 1. Estructura de una etiqueta

Las etiquetas HTML son los bloques de construcción básicos de una página web. Veamos su estructura:

<Tabs>
  <TabItem label="Explicación">
    Una etiqueta HTML típicamente consta de:
    - Una etiqueta de apertura: `<nombre-etiqueta>`
    - Contenido
    - Una etiqueta de cierre: `</nombre-etiqueta>`
    
    Algunas etiquetas son "auto-cerrantes" y no requieren una etiqueta de cierre separada.
  </TabItem>
  <TabItem label="Ejemplo">
    ```html
    <p>Este es un párrafo.</p>
    <img src="imagen.jpg" alt="Una imagen" />
    ```
  </TabItem>
</Tabs>

## 2. Estructura de una página web

Cada página HTML sigue una estructura básica. Veamos cómo se compone:

<Steps>
  1. Declaración del tipo de documento (`<!DOCTYPE html>`)
  2. Elemento `<html>` que envuelve todo el contenido
  3. Elemento `<head>` para metadatos
  4. Elemento `<body>` para el contenido visible
</Steps>

```html
<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Mi Primera Página Web</title>
</head>
<body>
    <h1>Bienvenido a mi sitio web</h1>
    <p>Este es un párrafo de ejemplo.</p>
</body>
</html>
```

## 3. Párrafos y encabezados

Los párrafos y encabezados son elementos fundamentales para estructurar el contenido de tu página.

<Tabs>
  <TabItem label="Párrafos">
    Los párrafos se crean con la etiqueta `<p>`. Ejemplo:
    ```html
    <p>Este es un párrafo de ejemplo. Puede contener múltiples oraciones.</p>
    ```
  </TabItem>
  <TabItem label="Encabezados">
    Los encabezados van desde `<h1>` (el más importante) hasta `<h6>` (el menos importante). Ejemplo:
    ```html
    <h1>Título principal</h1>
    <h2>Subtítulo</h2>
    <h3>Encabezado de nivel 3</h3>
    ```
  </TabItem>
</Tabs>

## 4. Listas

HTML ofrece tres tipos de listas: ordenadas, desordenadas y de definición.

<CardGrid>
  <Card title="Listas ordenadas" icon="list-ordered">
    Usa `<ol>` para crear listas numeradas.
  </Card>
  <Card title="Listas desordenadas" icon="list-unordered">
    Usa `<ul>` para listas con viñetas.
  </Card>
  <Card title="Listas de definición" icon="list">
    Usa `<dl>` para listas de términos y definiciones.
  </Card>
</CardGrid>

Ejemplo de lista ordenada y desordenada:

```html
<ol>
  <li>Primer elemento</li>
  <li>Segundo elemento</li>
</ol>

<ul>
  <li>Elemento de lista</li>
  <li>Otro elemento</li>
</ul>
```

## 5. Enlaces (Básico)

Los enlaces son fundamentales en HTML, ya que permiten la navegación entre páginas web. Se crean con la etiqueta `<a>`.

<Aside type="tip">
  El atributo más importante de un enlace es `href`, que especifica la URL de destino.
</Aside>

Ejemplo de enlace:

```html
<a href="https://www.ejemplo.com">Visita Ejemplo.com</a>
```

## 6. Imágenes y Rutas

Las imágenes se insertan en HTML usando la etiqueta `<img>`. Es una etiqueta de auto-cierre, lo que significa que no necesita una etiqueta de cierre separada.

<Tabs>
  <TabItem label="Sintaxis básica">
    ```html
    <img src="ruta/a/la/imagen.jpg" alt="Descripción de la imagen">
    ```
  </TabItem>
  <TabItem label="Atributos importantes">
    - `src`: La ruta a la imagen
    - `alt`: Texto alternativo para accesibilidad
    - `width` y `height`: Dimensiones de la imagen (opcional)
  </TabItem>
</Tabs>

### Tipos de rutas:

1. **Rutas absolutas**: Apuntan a una ubicación específica en la web.
   ```html
   <img src="https://www.ejemplo.com/images/foto.jpg" alt="Foto de ejemplo">
   ```

2. **Rutas relativas**: Apuntan a una ubicación relativa al archivo HTML actual.
   ```html
   <img src="images/foto.jpg" alt="Foto de ejemplo">
   ```

## 7. Formularios

Los formularios en HTML permiten recopilar datos de los usuarios. Se crean con la etiqueta `<form>` y contienen varios tipos de elementos de entrada.

<Steps>
  1. Crear un formulario con `<form>`
  2. Añadir elementos de entrada como `<input>`, `<textarea>`, `<select>`
  3. Usar etiquetas `<label>` para describir los campos
  4. Incluir un botón de envío
</Steps>

Ejemplo de un formulario básico:

```html
<form action="/enviar-datos" method="post">
  <label for="nombre">Nombre:</label>
  <input type="text" id="nombre" name="nombre" required>
  
  <label for="email">Email:</label>
  <input type="email" id="email" name="email" required>
  
  <label for="mensaje">Mensaje:</label>
  <textarea id="mensaje" name="mensaje"></textarea>
  
  <button type="submit">Enviar</button>
</form>
```
