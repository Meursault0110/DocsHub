## 2. Textarea & Labels

Los elementos `<textarea>` y `<label>` son fundamentales para crear formularios más complejos y accesibles.

<Steps>
  1. Crear un `<textarea>` para entrada de texto multilínea
  2. Asociar un `<label>` con el `<textarea>` usando el atributo `for`
  3. Ajustar el tamaño del `<textarea>` con los atributos `rows` y `cols`
</Steps>

Ejemplo de uso:

```html
<label for="comentario">Deja tu comentario:</label>
<textarea id="comentario" name="comentario" rows="4" cols="50">
Escribe aquí tu comentario...
</textarea>
```

## 6. Enlaces (Avanzado)

Los enlaces en HTML pueden ser más sofisticados que simples vínculos de texto. Aquí hay algunas técnicas avanzadas:

<CardGrid>
  <Card title="Enlaces internos" icon="link">
    Navega a secciones específicas de la misma página.
  </Card>
  <Card title="Atributos de destino" icon="open">
    Controla cómo se abre el enlace.
  </Card>
  <Card title="Enlaces de descarga" icon="download">

  </Card>
  <Card title="Enlaces de correo electrónico" icon="email">

  </Card>

Ejemplos:

```html

<!-- Enlace de descarga -->

<!-- Enlace de correo electrónico -->

```

## 7. Tablas

Las tablas en HTML se utilizan para presentar datos en filas y columnas. Aquí hay una estructura básica de una tabla:

<Steps>
  1. Usar `<table>` para crear la tabla
  2. Definir filas con `<tr>`
  3. Crear encabezados con `<th>`
  4. Insertar datos en celdas con `<td>`
  5. Utilizar `<caption>` para un título descriptivo
</Steps>

Ejemplo de una tabla HTML:

```html
<table>
  <caption>Horario de Clases</caption>
  <thead>
    <tr>
      <th>Hora</th>
      <th>Lunes</th>
      <th>Martes</th>
      <th>Miércoles</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>9:00</td>
      <td>Matemáticas</td>
      <td>Historia</td>
      <td>Ciencias</td>
    </tr>
    <tr>
      <td>10:00</td>
      <td>Literatura</td>
      <td>Física</td>
      <td>Arte</td>
    </tr>
  </tbody>
</table>
```

## 9. Lazy Loading

El lazy loading es una técnica que retrasa la carga de imágenes y otros recursos hasta que sean necesarios, mejorando el rendimiento de la página.

```html
<img src="imagen.jpg" loading="lazy" alt="Descripción de la imagen">
```

<Aside type="tip">
  El atributo `loading="lazy"` es compatible con la mayoría de los navegadores modernos y ayuda a mejorar significativamente el tiempo de carga de las páginas con muchas imágenes.
</Aside>

## 11. HTML Semántico

El HTML semántico utiliza etiquetas que transmiten significado sobre la estructura y el contenido de la página.

<Steps>
  1. Usar `<header>` para encabezados de página o sección
  2. Emplear `<nav>` para menús de navegación
  3. Utilizar `<main>` para el contenido principal
  4. Implementar `<article>` para contenido independiente
  5. Usar `<aside>` para información relacionada pero no esencial
  6. Emplear `<footer>` para pies de página
</Steps>

Ejemplo de estructura semántica:
