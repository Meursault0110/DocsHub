## Introducción a Grid

CSS Grid Layout es un sistema de diseño bidimensional que revoluciona la forma en que creamos estructuras en la web. Permite dividir una página en filas y columnas, creando diseños complejos con facilidad.

<Card title="¿Por qué usar Grid?" icon="puzzle">
  - Diseños bidimensionales
  - Control preciso sobre filas y columnas
  - Simplifica estructuras complejas
  - Responsive design nativo
</Card>

## Crear un Grid

Para comenzar a usar Grid, simplemente define `display: grid` en el contenedor padre.

```css
.container {
  display: grid;
  grid-template-columns: 200px 200px 200px;
  grid-template-rows: 100px 100px;
}
```

<Aside type="note">
  Este código crea un grid de 3 columnas y 2 filas.
</Aside>

## Unidades auto y fr

Las unidades `auto` y `fr` son cruciales en Grid para crear diseños flexibles.

<Tabs>
  <TabItem label="auto">
    `auto` ajusta el tamaño de la columna o fila al contenido.
  </TabItem>
  <TabItem label="fr">
    `fr` (fracción) distribuye el espacio disponible proporcionalmente.
  </TabItem>
</Tabs>

```css
.container {
  display: grid;
  grid-template-columns: 1fr auto 2fr;
}
```

## repeat() y minmax()

`repeat()` y `minmax()` son funciones que hacen que Grid sea aún más poderoso y flexible.

<CardGrid>
  <Card title="repeat()">
    Repite un patrón de columnas o filas.
    ```css
    grid-template-columns: repeat(3, 1fr);
    ```
  </Card>
  <Card title="minmax()">
    Define un rango de tamaño entre un mínimo y un máximo.
    ```css
    grid-template-columns: minmax(100px, 1fr) 2fr 1fr;
    ```
  </Card>
</CardGrid>

## Grid explícito e Implícito

Grid nos permite definir estructuras explícitas e implícitas.

<Tabs>
  <TabItem label="Explícito">
    Definido con `grid-template-columns` y `grid-template-rows`.
  </TabItem>
  <TabItem label="Implícito">
    Creado automáticamente cuando el contenido excede el grid explícito.
  </TabItem>
</Tabs>

```css
.container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  grid-auto-rows: minmax(100px, auto);
}
```

<Aside type="caution">
  El grid implícito puede llevar a diseños inesperados si no se maneja correctamente.
</Aside>

## Grid gap

`grid-gap` (o `gap` en versiones más recientes) define el espacio entre filas y columnas.

```css
.container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 20px;
}
```

## Grid dinámico (Responsive)

Grid es naturalmente responsive, pero podemos mejorar su adaptabilidad con media queries y funciones como `minmax()`.

```css
.container {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(200px, 1fr));
  gap: 20px;
}
```

<Aside type="tip">
  `auto-fit` y `auto-fill` son excelentes para crear layouts que se adaptan automáticamente.
</Aside>

## Grid column y grid row

Estas propiedades permiten a los elementos hijos ocupar múltiples celdas.

<Steps>
  1. Define el grid en el contenedor
  2. Usa `grid-column` y `grid-row` en los hijos
  3. Especifica inicio y fin de la expansión
</Steps>

```css
.item {
  grid-column: 1 / 3;
  grid-row: 1 / 3;
}
```

## Grid flow - Dense

`grid-auto-flow: dense` optimiza la colocación de elementos para llenar espacios vacíos.

```css
.container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  grid-auto-flow: dense;
}
```

<Badge text="Avanzado" variant="caution" />

## Grid Areas

Grid Areas permite nombrar secciones del grid y usarlas para posicionar elementos.

```css
.container {
  display: grid;
  grid-template-areas:
    "header header header"
    "sidebar main main"
    "footer footer footer";
}

.header { grid-area: header; }
.sidebar { grid-area: sidebar; }
.main { grid-area: main; }
.footer { grid-area: footer; }
```

## Alineación en Grid

Grid ofrece poderosas opciones de alineación tanto para el contenedor como para los elementos individuales.

<CardGrid>
  <Card title="Contenedor">
    - `justify-content`
    - `align-content`
  </Card>
  <Card title="Elementos">
    - `justify-self`
    - `align-self`
  </Card>
</CardGrid>

```css
.container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  justify-content: center;
  align-content: center;
}

.item {
  justify-self: end;
  align-self: center;
}
```

## Subgrid

Subgrid es una característica más reciente de CSS Grid que permite a los elementos hijos de un grid heredar la estructura de su padre.

<Badge text="Nuevo en CSS Grid Level 2" variant="note" />

```css
.container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 20px;
}

.item {
  display: grid;
  grid-template-columns: subgrid;
}
```
