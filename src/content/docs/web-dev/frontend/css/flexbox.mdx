## Introducción a Flexbox

Flexbox es un modelo de diseño en CSS que permite crear layouts flexibles y responsivos de manera eficiente. Es especialmente útil para distribuir el espacio entre elementos en un contenedor, incluso cuando sus tamaños son desconocidos o dinámicos.

<Card title="Ventajas de Flexbox" icon="rocket">
  - Simplifica la creación de diseños complejos
  - Facilita la alineación de elementos
  - Permite un control preciso sobre el espacio y el orden de los elementos
  - Mejora la responsividad del diseño
</Card>

## Flex Wrap y Flex Flow

### Flex Wrap

La propiedad `flex-wrap` determina si los elementos flex deben envolverse o no cuando no caben en una sola línea.

<Tabs>
  <TabItem label="nowrap">
    Los elementos se mantienen en una sola línea (valor por defecto).
  </TabItem>
  <TabItem label="wrap">
    Los elementos se envuelven en múltiples líneas si es necesario.
  </TabItem>
  <TabItem label="wrap-reverse">
    Los elementos se envuelven en múltiples líneas en orden inverso.
  </TabItem>
</Tabs>

### Flex Flow

`flex-flow` es una propiedad abreviada que combina `flex-direction` y `flex-wrap`.

```css
.container {
  flex-flow: row wrap;
}
```

## Order

La propiedad `order` permite cambiar el orden de los elementos flex sin modificar el HTML.

<Steps>
  1. Por defecto, todos los elementos flex tienen un `order` de 0.
  2. Los valores más bajos se colocan antes que los valores más altos.
  3. Los elementos con el mismo valor de `order` se ordenan según su posición en el HTML.
</Steps>

```css
.item1 { order: 2; }
.item2 { order: 1; }
.item3 { order: 3; }
```

## Flex Grow (Flex-items)

`flex-grow` define la capacidad de un elemento flex para crecer si es necesario. Acepta un valor numérico sin unidades que sirve como proporción.

<Aside type="note">
  Si todos los elementos tienen `flex-grow: 1`, ocuparán el mismo espacio. Si un elemento tiene `flex-grow: 2`, ocupará el doble de espacio que los otros.
</Aside>

```css
.item1 { flex-grow: 1; }
.item2 { flex-grow: 2; }
.item3 { flex-grow: 1; }
```

## Flex Shrink (flex-items)

`flex-shrink` define la capacidad de un elemento flex para encogerse si es necesario. Funciona de manera similar a `flex-grow`, pero en la dirección opuesta.

<Aside type="tip">
  Un valor mayor de `flex-shrink` hará que el elemento se encoja más en comparación con otros elementos cuando no haya suficiente espacio en el contenedor.
</Aside>

```css
.item1 { flex-shrink: 1; }
.item2 { flex-shrink: 2; }
.item3 { flex-shrink: 1; }
```

## Flex Basis (flex-items)

`flex-basis` define el tamaño predeterminado de un elemento antes de que se distribuya el espacio restante. Puede ser una longitud (e.g., 20%, 5rem, etc.) o una palabra clave.

<Tabs>
  <TabItem label="auto">
    El tamaño se basa en el contenido del elemento (valor por defecto).
  </TabItem>
  <TabItem label="0">
    El elemento no tiene tamaño inicial, y se distribuirá según `flex-grow`.
  </TabItem>
  <TabItem label="Longitud">
    Especifica un tamaño inicial fijo para el elemento.
  </TabItem>
</Tabs>

```css
.item1 { flex-basis: auto; }
.item2 { flex-basis: 0; }
.item3 { flex-basis: 200px; }
```

## Flex (shorthand)

La propiedad `flex` es una abreviatura para `flex-grow`, `flex-shrink` y `flex-basis` combinados. El segundo y tercer parámetros (`flex-shrink` y `flex-basis`) son opcionales.

<Aside type="tip">
  Se recomienda usar esta propiedad abreviada en lugar de establecer las propiedades individuales, ya que especifica inteligentemente los valores predeterminados.
</Aside>

```css
.item {
  flex: 1 1 auto; /* flex-grow: 1, flex-shrink: 1, flex-basis: auto */
}
```

Algunos valores predefinidos comunes:

<CardGrid>
  <Card title="flex: initial">
    Equivalente a `flex: 0 1 auto` (tamaño basado en el contenido, puede encoger)
  </Card>
  <Card title="flex: auto">
    Equivalente a `flex: 1 1 auto` (tamaño basado en el contenido, puede crecer y encoger)
  </Card>
  <Card title="flex: none">
    Equivalente a `flex: 0 0 auto` (tamaño basado en el contenido, no puede crecer ni encoger)
  </Card>
  <Card title="flex: <positive-number>">
    Equivalente a `flex: <positive-number> 1 0` (puede crecer y encoger, sin base inicial)
  </Card>
</CardGrid>

## Gap (flex-items)

La propiedad `gap` establece los espacios (gutters) entre filas y columnas. Es una abreviatura para `row-gap` y `column-gap`.

```css
.container {
  display: flex;
  gap: 10px; /* Espacio de 10px entre todos los elementos */
}
```
